quiz-plugin

This plugin enables the usage of moodle-quizzes in QGIS

What you can do:

  1. Take normal moodle tests in QGIS
  2. Train quizzes for optimal progress
  3. Import moodle-quizzes from anywhere on your harddrive

    Supported questiontypes:

    1. Multiple choice
    2. Missing word / Fill in the gap
    3. Matching
    4. Questions with pictures
    5. Shortanswers
